# QR Code Generator

Este é um site desenvolvido para criar e gerar QR Codes de forma rápida e fácil. O website oferece uma interface simples e responsiva, permitindo que os usuários criem códigos QR em poucos cliques.

## Sobre
O site permite que os usuários gerem QR Codes personalizados para links, textos, ou outras informações, com uma experiência intuitiva e eficiente.

## Tecnologias
- HTML

## Como usar
```bash
git clone https://github.com/euoryan/qrcode
cd qrcode
```
